## 🎨 改善提案（優先度別）

## 🔥 優先度：HIGH

### 1. ネオンパーティクルエフェクト（背景）
**実装箇所**: 全画面の背景レイヤー

```tsx
// 小さな光の粒子がゆっくり上昇・浮遊
// cyan, blue, purple, pinkの4色をランダムで配置
// 20-30個の粒子が常時アニメーション
```

**効果**:
- 静的な背景が「生きている」印象に
- 深夜のバーの幻想的な雰囲気
- **実装コスト**: 中（Canvas or CSS animation）

---

### 2. グローパルス（呼吸するグロー）
**実装箇所**:
- OPENボタン
- CTAボタン（「この客にする」「提供する」など）
- 選択中のカード

```css
@keyframes glow-pulse {
  0%, 100% {
    box-shadow: 0 0 20px rgba(76, 199, 255, 0.5);
  }
  50% {
    box-shadow: 0 0 40px rgba(76, 199, 255, 0.8);
  }
}
```

**効果**:
- 注目を引く要素が「呼吸」しているように見える
- ボタンの押したくなる感を強化
- **実装コスト**: 低（CSS keyframes）

## 🚨 注意点

### パフォーマンス
- モバイルでは重いエフェクトを軽量化
- `will-change` の適切な使用
- アニメーションのFPS監視

### アクセシビリティ
- `prefers-reduced-motion` の考慮
- 過度なアニメーションによる酔い防止
- オプションでエフェクトON/OFF
